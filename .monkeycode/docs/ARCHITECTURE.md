## 概述

AI Code Review Platform是一个基于AI的代码审查平台，提供智能代码分析、架构合规性检查和安全漏洞扫描等功能。

### 核心特性

- AI驱动的代码审查
- Pull Request自动分析
- 架构合规性检查
- 安全漏洞扫描
- 实时协作
- GitHub集成
- 5级RBAC权限控制

### 分层说明

#### 1. 用户界面层

- **Web前端**：基于Next.js的单页应用
- **API网关**：统一API入口，处理请求路由
- **响应式设计**：支持多设备访问

#### 2. 应用层

- **认证授权**：基于RBAC的权限管理
- **业务服务**：各种业务逻辑服务
- **AI分析**：AI驱动的代码分析服务

#### 3. 数据层

- **PostgreSQL**：主要数据存储
- **Redis**：缓存和会话管理
- **Neo4j**：图数据库，存储代码依赖关系

#### 4. 外部集成

- **GitHub API**：代码仓库集成
- **LLM服务**：AI模型调用
- **监控系统**：性能监控和告警

## 数据流

### 代码审查流程

```
用户提交PR
    ↓
GitHub Webhook触发
    ↓
拉取代码变更
    ↓
代码分析
    ├─ 静态分析
    ├─ 安全扫描
    └─ AI分析
    ↓
生成审查报告
    ↓
保存到数据库
    ↓
通知用户
    ↓
用户查看报告
```

### 认证流程

```
用户登录
    ↓
提交凭证
    ↓
验证凭证
    ├─ 本地数据库
    └─ GitHub OAuth
    ↓
生成JWT令牌
    ↓
返回令牌
    ↓
后续请求携带令牌
    ↓
验证令牌
    ├─ 检查签名
    └─ 检查过期
    ↓
授权访问
```

## 架构决策记录

### ADR-001: 选择Next.js作为前端框架

**决策**：使用Next.js构建前端应用
**原因**：

- 服务端渲染提升SEO
- 自动代码分割优化性能
- 内置路由和API路由
- 强大的生态系统

**替代方案**：React SPA, Vue.js, Angular

### ADR-002: 选择FastAPI作为后端框架

**决策**：使用FastAPI构建后端API
**原因**：

- 高性能（异步支持）
- 自动生成API文档
- 类型安全（Pydantic）
- 现代Python特性

**替代方案**：Django, Flask, Express.js

### ADR-003: 选择PostgreSQL作为主数据库

**决策**：使用PostgreSQL存储核心数据
**原因**：

- 强大的关系查询能力
- 支持复杂事务
- 丰富的数据类型
- 成熟稳定

**替代方案**：MySQL, MongoDB

### ADR-004: 集成Neo4j图数据库

**决策**：使用Neo4j存储代码依赖关系
**原因**：

- 天然支持图数据
- 高效的图查询
- 直观的数据模型
- 适合代码分析

**替代方案**：PostgreSQL (扩展), ArangoDB
